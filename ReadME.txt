Part.4 一些说明 
       A.关于客户端
       	请尽量使用 Java 8 (比较稳定)[ 如果出于某些原因无法使用Java8 请尽量使用高于 Java8 的版本]
       	请不要随意更改json文件
       	除mods/shaderpacks/resourcepacks文件夹内文件请不要随意增删
       
       B.关于服务器
	正式开服时间是我开学后
	服务器内物品掉落后消失时间为 3 分钟
	在服务器内禁止使用核弹但可少量使用TNT【我不想装插件去限制这些东西，因为懒】
	高频红石电路请尽量少用 能不用就不用 如果需要脉冲 请尽量使用低频红石电路
	请不要4人以上同时进入暮色森林 要不然我这内存占用会出现波动 导致服务器不稳定
	视距默认为 8Chunks 无法自行修改
	会考虑服务器多世界生存（大坑）

Part.6 一些教程
	1. BetterFonts [更好的字体]
		在主界面点击 Mods 进入Mods列表
		在其中利用搜索或者手动找到 BetterFonts 模组并选中它
		单击 Config 按钮 之后你会看到以下内容
		     |    Anti-Alias         抗锯齿 (AA!)
		     |    DropShadow  	     阴影
		     |    fontName           字体
	  	     |    fontSize           字号
		按照上述说明进行更改即可

	2. JustEnoughItems [JEI]
		直接在游戏里按E键召唤物品栏 右侧会有GUI显示的物品 单击其图标即可召唤合成
	
	3.CustomOreGen [ 矿物生成自定义 ]
		在单机游戏时，生成地图设置时，有CunstomOreGeneration的选项进去之后完全汉化 可以根据你的需要来选择  【注意 服务器的矿物生成是根据服务器的矿物配置 但需要此Mods读取设置】
