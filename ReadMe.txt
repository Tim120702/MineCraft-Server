放到.minecraft/versions/<你的版本号>文件夹下【仅适用于BakaXL等强制版本分离的启动器】
【还有一些启动器不强制版本分离你找到.minecraft/shaderpacks就可以了】

启动游戏，光影设置在##选项->视频设置->光影##中调整

推荐SEUS v10.1光影，效果不错

Part.3 关于皮肤

	若果你想要使用皮肤, 请先给@Bikov 提供你登陆客户端时所使用的用户名【注2】(无论正版盗版)
	当然 正版用户对用户名有优先占有权 ！！！
	如果有重名或不恰当名字 我会提醒你 请尽快修改
	用户名登记成功后 可以使用你的用户名 + 群内密码登录皮肤站 ( 仅提供皮肤搜寻或更改 )

	请优先在群内上传的 LittleSkin 皮肤站上的皮肤库内寻找你心仪的皮肤

	如果没有你想要的皮肤，你可以自行从资源站【注1】寻找皮肤下载或自己制作，然后上传至 
	LittleSkin 皮肤站 (请注意皮肤站皮肤上传规则 )，之后根据网站指引创建自己的用户名并应用
	相应皮肤。（网站有相应指南，请先花几分钟阅读一下）

	如果你无法操作，也可以在群内@Bikov，并提交自己的用户名和相应皮肤文件。
	如果你不能下载皮肤文件，可以提交相应网站URL(并指明你要的皮肤) 给@Bikov

	如果你提交的皮肤无法上传 我会提醒你你可以重新寻找新的符合条件的皮肤【注3】
	如果你想更换皮肤 请重复上述操作

		【注1】：可以使用的资源站  http://www.mcbbs.net/
				          http://skin.minecraftxz.com/ 
			#请不要从其他皮肤站找皮肤(eg.SkinMe) 因为上面的皮肤可能无法下载#

		【注2】：你使用BakaXL登陆时使用的用户名
		               #如果你的用户名不恰当 在你更改用户名前 你无法使用皮肤#

		【注3】：符合条件的皮肤如下（一般从资源站上下载的皮肤会符合相应条件）
			单层皮肤（大小为 64*32 或其整数倍）
			双层皮肤（大小为 64*64 或其整数倍）
			64*32 的披风（不支持 22*17 的披风）
			所有材质文件应该是 PNG 格式的图片。
			因为服务器版本问题（1.7.10）仅支持Steve皮肤不支持Alex皮肤

Part.4 关于服务器

       A.关于客户端
       	请尽量使用 Java 8 (比较稳定)
       	请不要随意更改json文件
       	除 mods/shaderpacks/resourcepacks 文件夹 其他文件夹内文件请不要随意增删
       
       B.关于服务器

	正式开服时间是我开学后【现在是删档公测】
	服务器内物品掉落后消失时间为 3 分钟
	在服务器内禁止使用核弹但可少量使用TNT
	高频红石电路能不用就不用 如果需要脉冲 请尽量使用低频红石电路

	会考虑服务器多世界生存（大坑）【已实现】
	正式开服会有权限自动分配
